# Car Notificator
Web service that serves to send notifications about the availability of new cars for sale.
## Used technologies
- Java 21
- Spring Framework
  - Spring Boot
  - Spring Security
  - Spring Data JPA
  - Spring MVC
  - Spring Test
- Apache Maven
- Hibernate
- PostgreSQL
- Apache Kafka
- Flyway
- Thymeleaf
- Tailwind CSS
